Install instructions:

1: Place the script folder inside the mod folder from your Victoria 2 install directory or if your using vanilla just inside the Victoria2 folder

	Steam version : C:/Program Files (x86)/Steam/steamapps/common/Victoria 2/mod/HPM(or other mod folder used for save game)
	non-Steam version : C:/Program Files (x86)/Paradox Interactive/Victoria 2/mod/HPM(or other mod folder used for save game)

2: Place your save game files you want the script to get the data from in the save games folder inside the script folder

3: Run “startscript.bat”

4: CSV files will be inside the outputs folder
